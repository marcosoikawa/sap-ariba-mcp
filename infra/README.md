# Infraestrutura para Deploy no Azure

Esta pasta contém os arquivos necessários para provisionar e implantar a solução SAP Ariba MCP + Agent no Azure usando o Azure Developer CLI (`azd`).

## Scripts e Arquivos
- `main.bicep` — Infraestrutura principal (opcional, pode ser substituído por templates do azd)
- `azure.yaml` — Definição do projeto para o Azure Developer CLI
- `env` — Exemplo de variáveis de ambiente

Consulte o README principal para instruções detalhadas de uso.
